# **App Name**: RideWise India

## Core Features:

- Smart Location Input: Location input with current location detection and manual address entry for origin and destination.
- Fare Comparison Display: Comparison table/cards displaying Uber, Rapido, and Ola fares, ETAs, and total travel time.
- Detailed Fare Breakdown: Price breakdown for each provider, including base fare, distance/time charges, taxes, and surge pricing.
- Seamless App Redirection: On-tap redirect to selected ride-hailing app with pre-filled pickup and drop location.
- Predictive Fare Analysis: AI-powered tool for fare prediction during peak hours and suggesting optimal times to book.
- Cheapest/Fastest Indicators: Visual highlighting of the cheapest and fastest options.

## Style Guidelines:

- Primary color: Deep saffron (#FF9933) to represent the vibrancy of India, contrasting against a dark or light background.
- Background color: Light scheme: Off-white (#FAF9F6); Dark scheme: Dark gray (#333333), both with low saturation.
- Accent color: Teal (#30D5C8), an analogous hue to saffron, used for interactive elements and highlighting key information, creating high contrast.
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text.
- Custom, recognizable icons for Uber, Rapido, and Ola to facilitate quick identification.
- Card-based layout for displaying ride options with clear separation and a clean, modern look.
- Smooth transitions and animations for a user-friendly experience when comparing fares.
